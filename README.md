# HealthSaathi 🩺

A simple daily health tracker web app, built with the idea of eventually adding **Indian-context-aware AI coaching** — nudges and suggestions that actually make sense for daily life in India (food habits, routines, etc.), rather than generic fitness-app advice.

This is a personal learning project built to explore frontend development and, eventually, AI integration — from scratch.

## What it does

- Lets users log and track daily health habits (e.g. water intake, meals, activity — customize this line to match what your tracker actually logs)
- Clean, simple UI built with plain HTML, CSS, and JavaScript — no frameworks
- Designed with a future AI coaching layer in mind, tailored to Indian users

## Tech Stack

- **HTML** — structure
- **CSS** — styling
- **JavaScript** — app logic and interactivity

## Status

🚧 Work in progress. Core tracker UI is functional. Next steps include:
- Adding AI-powered coaching suggestions (exploring OpenRouter / Gemini APIs)
- Visual/UI improvements

## Running it locally

1. Clone the repo
   ```bash
   git clone https://github.com/Shuhul-tickoo/healthsaathi.git
   ```
2. Open `index.html` in your browser (or use a Live Server extension in VS Code)
